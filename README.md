Morse code is a method of commumication that uses a series of dots(.) and dashes(-) to represent letters, numbers, symbols. it was mainly used for sending messages over long distances using sound, light, or radio signals.

Morse code was developed in the 1830s and 1840s by samuel Morse and Alfred Vail. it was created to be used with the telegraph system, which allowed messages to be sent quickly over wires. This invention changed communication by making long-distance messaging faster and more efficient.

Morse Code uses timing.
dots(.) short signals
Dashes(-) long signals
space between dots/dashes in same letter =1 unit
space between letters = 3 units
