Morse code is a method of commumication that uses a series of dots(.) and dashes(-) to represent letters, numbers and punctuation. Instead of using written text or spoken words, messages are transmitted through signals such as sound, light, or electrical pulses. Each character in the alphabet has a unique combination of dots and dashes, allowing people to encode and decode messages.

Morse code was developed in the 1830s and 1840s by samuel Morse and Alfred Vail. it was created to be used with the telegraph system, which allowed messages to be sent quickly over wires. It was used to support long distance communication. At the time, the was a need for a fast and reliable way to send messages over wires across great distance. The system made it possible to transmit information efficiently using simple electrical signals, which played a major role in communication during that era.

Morse Code uses timing.
dots(.) last for a short time
Dashes(-) last about three times longer than a dot
The space between letter is a bit longer
The space between words is even longer
The timing helps people tell the difference between letters and words
